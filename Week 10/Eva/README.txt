- original_data/ contains the MSA and metadata file downloaded from the ncbi virus database.
- MSA.fasta and sequence_metadata.csv are the processed MSA and metadata files for usage with TreeTime and Nextatrain augur.
- UPGMA_tree.png shows the phylogenetic tree produced in Task 2.
- augur_out/ contains the input and results from analysing the data using Nextstrain augur and auspice. augur_out/auspice/ contains the JSON file that was visualized using Nextstrain auspice. 
- the jupyter notebook implements or in the case of Nextstrain augur and auspice reports all analysis steps that were performed within the scope of this project
